# StoryNest
A decentralized platform for creating and sharing short animated stories on the Stacks blockchain.

## Features
- Create story NFTs with metadata
- Share stories with other users
- Like and collect stories
- Earn rewards for popular stories
- Trade stories on the marketplace

## Contract Overview
- `story-nft.clar`: Main NFT contract for story creation and ownership
- `marketplace.clar`: Handles story trading and transactions
- `rewards.clar`: Manages creator rewards system
